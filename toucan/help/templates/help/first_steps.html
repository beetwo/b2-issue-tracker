Here are just a few small tips to make your first steps with Toucan a smoother experience.

Get notified of issues near you

I want to be notified of medical issues within 20 km of my home.

When people and organisations create issues on Toucan they are usually looking for help.
With Toucan you can be notified when new a issue is created that matches your criteria. Since all issues inside Toucan are location specific the most important way to find issues that you might be able to resolve is through the use of this location information.
Currently Toucan supports notifications within a certain radius of a point.

Add a notification rule

Get notifications when you are mentioned

I want to be notified when I am mentioned in comments.

A big part of the value proposition of Toucan is that you can reach people and organisations that would otherwise not be able see your requests for help.
Of course there are always cases where you might want to address people/organisations directly and for this reason Toucan has a user notification system built-in. You as a user can decide if and how you want to be notified of certain actions that happen in Toucan.

Go to notification settings

Add an issue

Issues in Toucan can be created by either right-clicking the map at the desired location or following the 'Add issue' in the main menu. Both ways will bring you to an issue creation form where you will need to fill out a few fields. After saving the new issue will be visible to every user on Toucan.
